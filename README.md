## ✨ Features

- **Modern Glassmorphic Design:** Implements high-end CSS aesthetics featuring radial gradients, subtle translucent backgrounds (`backdrop-filter`), smooth border shadows, and a blue glowing focus aura on hover.
- **Pure CSS Interactivity:** Separates presentation from behavior by migrating hover animations, transform transitions, and image rotations completely into CSS rules rather than JavaScript.
- **Responsive Layout Architecture:** Adaptive design built using modern Flexbox layouts. Adapts fluidly down to mobile viewports (`320px` width) with responsive columns that transition to a stacked single-column layout on smaller screens.
- **Semantic & Accessible (HTML5):** Employs standard semantic landmarks (`<main>`) and converts generic elements into accessible anchor links with proper anchor attributes (`target="_blank"`, `rel="noopener noreferrer"`).
- **Embedded SVG Iconography:** Integrates lightweight, crisp inline SVG icons next to actions to provide modern, professional visual cues.
- **SEO & Social Optimization:** Features custom SEO tags, author indexing, and complete Open Graph properties (`og:title`, `og:description`, `og:image`) to display clean rich snippets when shared on social sites like LinkedIn.

---

## 🛠️ Tech Stack

- **Markup:** Semantic [HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML) structures for improved accessibility and crawler search visibility.
- **Styling:** [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS) with custom CSS variables (design tokens), hover states, keyframe transforms, and glassmorphic blurs.
- **Behavior:** Vanilla [JavaScript (ES6+)](https://developer.mozilla.org/en-US/docs/Web/JavaScript) for profile opening actions and formatted URL mail client integration.
- **Typography:** [Google Fonts](https://fonts.google.com/) (`Outfit` for crisp, geometric hierarchy).

## ⚙️ Installation & Usage

Since this is a lightweight, zero-dependency frontend card, no build systems or dependencies are required.

### 1. Clone the Repository
```bash
git clone https://github.com/imzohaib-web/business-card.git
cd business-card
```

### 2. View Locally
* **Direct Execution:** Double-click the [index.html](index.html) file to open it instantly inside any web browser.
* **Development Server (Recommended):** Right-click [index.html](index.html) and open with the **Live Server** extension in VS Code, or launch a local preview using `npx serve .` in your command terminal.
